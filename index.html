ezMemeQuote

A Template Engine for Meme-esque Quotes

Text-based meme template, with a make-it-brand-centric twist. Some light CSS (and js) knowledge required.

When tweeting please use #ezMemeQuote.

Begs to be forked and tricked out.

Get ezMemeQuote on GitHub

Overview

The concept is simple; have a web designer (i.e., someone who can handle visual aesthetics and CSS) layout a brand-centric design template:

  • Text content is updated by the user / client via a JSON file (read: a text file);
  • Use a browser add-on / extension for the screen grab;
  • Finally, post that image to Pinterest, Facebook, Twitter and the like.

It's not Photoshop! It's a quick KISS tool for publishing branded quotes with minimal friction. That's sbout it.

Key Benefits & Features:

  • Local
    No website necessary. Everything happens in the local browser.

  • KISS
    Less is more, and ezMemeQuote is the mostest of the leastest.

  • Flexible
    Multiple style options can be CSS'ed and then selected by the user / client.

  • Brand-centric
    Define a consistent look that even a bat can stick to.

  • Text-based
    The content is stored as data in a JSON file so it's easy to reuse, find, search against, etc.

What it doesn't do:

  • Images and anything Abobe-ish - Sure you can have an image background but this isn't about images per se. The need was to generate branded meme-esque quotes with minimal effort and maximum results. Admittedly, the scope here is pretty tight. That sweet spot may or may not work for you.

    Clone it / fork it and give it an hour, then make your decision.

How To

Once you clone it / fork it look for the file: product / checklist / TODO.txt.

If you need help TODO.txt is the best place to start.

In addition, each individual is file is small, focused and typically well commented. If you've done some work with HTML, CSS and a bit of Javascript doesn't scare you, then you'll be comfortable with this.

The key bit to understand is that once everything is setup, only the file: product / a-edit-this / my-quotes.js gets edited.
